# Joystick

Publish [JOYSTICK](https://docs.m5stack.com/en/unit/joystick) state.

## Overview

Published topics:

- `/joy` (`sensor_msgs/Joy`)

  State of joystick

## Usage

- Follow [README.md](https://github.com/jsk-ros-pkg/jsk_3rdparty/tree/master/m5stack_ros)

- Run

  ```bash
  roslaunch m5stack_ros m5stack_ros.launch
  ```
