# MPU9250 with battery

This firmware is an addition to the [MPU9250](https://github.com/jsk-ros-pkg/jsk_3rdparty/tree/master/m5stack_ros/sketches/MPU9250) firmware with battery monitoring functionality.

In `MPU9250_with_battery` version, battery infomation (`battery_level` and `is_charging` rostopic) are published in addition to IMU information.
